GitHub 

- créer un dossier git sur ~/
- placer les fichiers dans le dossier ~/git 

Dans le terminal : 

- (la premère fois) faire git init 
- faire cd ~/git
- faire git add nomdufichier.extension
- faire git commit -m "description Commit"
- (si première fois) faire git remote add origin 	
  https://github.com/user/lasource.git
- faire git push -u origin master 
- entrez les identifiants

Télécharger depuis le fork :

- faire git clone https://github.com/user/lasource.git
